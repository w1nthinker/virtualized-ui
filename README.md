# VirutalizedUI
Virtual List & Grid, that have all the properties one can dream of from UIListLayout & UIGridLayout.
Works on every viewport size.

Copy the code from this repo or get the .rbxm from the release.
Create an issue for feedback & improvements.

## Info
- made with & for the new type solver
- fully strict typed
- optimized & instant up to engine limits
